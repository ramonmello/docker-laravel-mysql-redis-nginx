## Versões

-   **Laravel** — 8.10.0
-   **Mysql** — 5.7
-   **Redis** — imagem redis:alpine
-   **Nginx** — 1.15.0-alpine
-   **Docker Composer** — 3
<br>

## Como usar

1 - Faça o clone do projeto
2 - Dentro da pasta do projeto execute o comando `docker-compose up -d` para subir o ambiente
